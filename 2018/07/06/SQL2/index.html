2、两种盲注

(0)、小前言

盲注的定义：sql盲注就是不直接显示搜索到的结果，而从其他方式来推断得出结果的sql注入。
盲注的类型：布尔型盲注和时间型盲注(第三类：报错型盲注)。
涉及函数：
1) substr() 用于截断字符串

substr(string,start,length)
string(必需)规定要返回其中一部分的字符串。
start(必需)规定在字符串的何处开始。
length(可选)规定被返回字符串的长度。

2) mid()函数

mid(striing,start,length)
string(必需)规定要返回其中一部分的字符串。
start(必需)规定开始位置（起始值是 1）。
length(可选)要返回的字符数。如果省略，则 mid() 函数返回剩余文本。

3) limit offset 与 limit 用于查询到多条记录时选取第几条，再猜取内容，不然只会猜取第一条

例:
语句1：select * from student limit 9,4
语句2：slect * from student limit 4 offset 9
//语句1和2均返回表student的第10、11、12、13行
//语句2中的4表示返回4行，9表示从表的第十行开始

(1)、布尔型盲注

  只有false和true两种状态，过程中需要猜测，直到猜到正确型为止。

布尔型盲注步骤
<1>判断是否有盲注点

  只要当SQL语句执行成功和失败的返回的页面存在某种固定差异，即存在布尔型盲注点。
输入1’ and ‘1’ = ‘1，返回true;
输入1’ and ‘1’ = ‘2，返回false，并且没有返回。

(2)、时间型盲注

  对一个命令只有一个固定的反应，如果是正确的就会等待一定时间再反应，如果是错误的就会立刻反应。

时间型盲注步骤
<1>判断是否有盲注点

  SQL语句执行成功和失败的返回的页面不存在差异，需要通过以下语句判断是否存在时间型盲注点。
$name=1’ and if(‘1’ = ‘1’,1, sleep(5)) –+ (不延迟返回为true)
$name=1’ and if(‘1’ = ‘2’,1, sleep(5)) –+ (延迟返回为false)
全句：select 列 from 表 where name = ‘1’ and if(‘1’ = ‘1’,1,sleep(5)) –+

<2>利用函数来搜集信息

全句：select 2列 from 表 where name = ‘’ or if( (select substring( user(),1,1 )=’r’),1, sleep(5)) –+’
